## Features

* Procedural dungeons.
* Procedural textures.
* Bosses.
* Memes.

## How to run using Gradle
In repo's game folder execute:
```
./gradlew run
```

### Known bugs
* Mobes sometimes may see you through the walls.
* Mobes can be stucked in the walls.
* There are strange "shadow wall" in the middle of the map.
